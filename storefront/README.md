# mm-store storefront

this is the next.js frontend for mariner market.

### features
- **app router**: modern next.js routing.
- **stripe**: embedded credit card checkout.
- **meilisearch**: instant product search.
- **tailwind**: utility-first styling.

### setup
1. `npm install`
2. `cp .env.local.template .env.local`
3. `npm run dev` (starts on port 8000)

### requirements
needs a running medusa backend on port 9000 to fetch data.
